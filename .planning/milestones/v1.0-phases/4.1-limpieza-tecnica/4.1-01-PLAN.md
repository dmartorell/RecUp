    En `src/js/ticket-modal.js` línea 220, cambiar:

    ```js
    if (!res.ok) throw { partial: data.uploaded > 0, data };
    ```

    por:

    ```js
    if (!res.ok) throw { partial: data.uploaded.length > 0, data };
    ```

    La API en `server/routes/attachment.js` devuelve `uploaded: results` donde `results` es un Array.     Solo modificar esta línea. No refactorizar nada más.
  </action>
